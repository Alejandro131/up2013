Упражнение 1
============

### Задача 1 ###

От клавиатурата се въвежда цяло число, съответстващо на изминалите секунди относно 00:00 часа. Пресметнете и изведете на екрана колко часа е в момента във формата hh:mm.

#### Примери ####

```c++
100000
// -> 03:46

86400
// -> 00:00
```
