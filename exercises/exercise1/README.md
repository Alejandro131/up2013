Упражнение 1
============

### Задача 1 ###

Напишете програма, която, по дадени координати на 3 точки в двумерна декартова координатна система, намира лицето
на триъгълника, който образуват.

#### Примери ####

```c++
0 0
0 3
4 0
// -> 6

0 0
0.25 0.25
0 0.25
// -> 0.03125
```
